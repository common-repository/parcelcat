== Description ==

This module provides integration with Parcelcat.com. Parcelcat is a smart logistics warehouse that enables an e-commerce store to outsource the storage, administration and dispatch of all goods to us.

You will see the stock in real time, we will sort the goods according to the orders and ship them to your customer.

Our Smooth Fulfillment warehousing system provides you:

- Shipment of goods to the warehouse
- Returns
- Storage of goods
- Delivery of goods to the client

All these processes can be tracked in the system.

You receive:

- Saving time and money
- Process management
- Business development
- Time verified couriers
- Convenience
- Continuous support with directly assigned manager
- Integrations for various e-shops

== Installation ==

1. Install and enable the plugin. For details about finding, installing and enabling Wordpress plugins, please address this link, https://wordpress.org/support/article/managing-plugins/#finding-and-installing-plugins-1
2. Go to the plugin's settings page. Find "Parcelcat" in Wordpress administration menu, click there.
3. Find Parcelcat project key in your Parcelcat account by clicking project settings.
4. In this module only settings section you can see "Project ID" text field. Paste project key there.
5. Save the plugin's settings by clicking "Save Changes" button.
6. Enjoy yourself!
